📊 Column Guide
Event
The name of the event. Click any row to open the full event analysis page with price history charts and section data.
Cat
Event category — FIFA, NBA, MLB, NFL, NHL, F1, Concert, etc.
Days
Days until the event. Red = 7 days or less, Gold = within 30 days. Sortable.
Date
The event date. Default sort is soonest first. Click column header to toggle.
Venue
The venue name. Use the venue search bar to filter by location.
Floor
Cheapest listing currently on StubHub. Your listing needs to beat this to sell first. Sortable.
ATP
Average Ticket Price — the mean across all active listings. Useful for pricing for margin.
Listings
Total active listings right now. High = more competition. Low = rising prices likely.
Score
VKT's proprietary Demand Score (1–99). From days to event, sell-through, floor trend, and volume momentum. 60+ = high, 35–59 = moderate, below 35 = low.
Vol Chg %
Listing volume % change for the selected time period (24h, 7d, or 14d). Green/negative = listings dropped (tickets selling). Red/positive = listings grew (more competition). Switch periods using the Vol Period dropdown above the table.
Qty Chg
Raw listing count change for the selected time period. Shows the exact number of listings added or removed — e.g. -42 means 42 fewer listings than the start of the period. Use the Vol Period dropdown to switch between 24h, 7d, and 14d.
Alert 🔔
Set a custom alert on any data point — Floor, ATP, Listings, Demand Score, 24h Vol %, or Days to Event. Choose > (rises above) or < (drops below) a number or percentage. Your alerts are saved in your browser and shown as a mini label on the bell icon.
